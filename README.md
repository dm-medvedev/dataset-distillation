# Dataset Distillation
The code was forked from the [project](https://github.com/SsnL/dataset-distillation) and changed by [Dmitry Medvedev](https://github.com/dm-medvedev).

## Prerequisites

### System requirements
- Python 3
- CPU or NVIDIA GPU + CUDA

### Dependencies
- ``torch >= 1.0.0``
- ``torchvision >= 0.2.1``
- ``numpy``
- ``matplotlib``
- ``pyyaml``
- ``tqdm``
